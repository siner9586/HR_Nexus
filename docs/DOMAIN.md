## Custom domain

When a domain is provided, add it in Vercel Domains.

Subdomain example:

```text
hr.example.com CNAME cname.vercel-dns.com
```

Apex domains depend on the DNS provider and may use A, ALIAS, or ANAME records following Vercel's instructions.

## China mainland access

Vercel default domains can be unstable from some China mainland networks. For stronger mainland access, evaluate备案域名, Cloudflare, Hong Kong/Singapore infrastructure, domestic cloud deployment, or private deployment.
